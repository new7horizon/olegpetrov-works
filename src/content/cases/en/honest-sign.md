## The challenge

Russia's Honest Sign law made unit-level traceability mandatory for dairy: every pack now needs a unique code that is printed, checked and reported to the state — or it can't legally be sold.

The hard part was *where* it had to happen. This was a plant that never stops — packaging lines of different ages and makers, a cold, wet, food-grade floor, and no room for downtime. The marking had to be added to equipment that was already running every day.

## What was done

Each packaging line was fitted with a printer to apply the code and a vision camera to read it back, then wired into new field power, networking and a server that reports to the state system.

The real work wasn't any single device — it was making old machines, new sensors and the software agree, and tuning it on a running line until every pack read correctly the first time. The job was delivered on site, hands-on, from survey to a stable line.

## Technologies

- **Marking** — VideoJet DataFlex 6330 thermal-transfer coders and LetsMark TIJ printers, mounted onto the existing packaging machines.
- **Vision** — Datalogic and Cognex industrial cameras reading each DataMatrix code inline.
- **On the product** — unique unit-level codes across milk, butter, sour cream and kefir.
- **Network & server** — managed switching, UPS and a server tying the lines, cameras and operator stations to the national system.
